# 🏗️ Admin Panel

A clean, fast, and scalable admin dashboard for managing projects, users, and system data — built with **React**, **Vite**, **Tailwind**, and **shadcn/ui**.

---

## 📦 Tech Stack

This project uses:

- ⚡ **Vite** – for lightning-fast development builds
- 🧠 **TypeScript** – strict typing for safer code
- 🧱 **React** – component-based UI
- 🎨 **Tailwind CSS** – utility-first styling
- 🧩 **shadcn/ui** – clean, accessible UI components

---

## 🚀 Getting Started (for Devs)

You can work with this codebase in any of the following ways:

### 🔧 Option 1: Local Dev Setup

Clone and run locally with Node.js:

## Commands

# Clone the repo
git clone <your_git_url>

# Move into the project folder
cd lovable-admin

# Install dependencies
npm install

# Start the development server
npm run dev
